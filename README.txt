# Teh Kita
Website profil dan pemesanan online untuk kedai minuman Teh Kita. Proyek ini dibangun untuk mendigitalisasi proses pemesanan melalui antarmuka keranjang belanja yang terintegrasi langsung dengan WhatsApp penjual.

## Fitur Utama
* Katalog Menu Interaktif
* Keranjang Belanja Dinamis 
* Checkout Otomatis ke WhatsApp beserta detail harga dan pesanan
* Desain Responsif untuk Mobile dan Desktop
* Notifikasi interaktif menggunakan Toastify JS

## Teknologi
* HTML5
* CSS3
* JavaScript
* Bootstrap 5
* Toastify JS
